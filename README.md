# KiddoFinance – Fun Money Manager for Kids

KiddoFinance is a playful budgeting and savings app designed to help kids learn about money in a simple, game-like way.

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting started

```bash
git clone <YOUR_GIT_URL>
cd kiddo-coin-capers
npm install
npm run dev
```

Then open your browser at `http://localhost:8080/` (or whatever port Vite prints in the terminal).

## Scripts

- `npm run dev` – start the development server
- `npm run build` – create a production build
- `npm run preview` – preview the production build locally
